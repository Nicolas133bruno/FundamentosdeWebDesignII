Instruções e Resultados em JavaScript

Instrução(ões) Resultado Observação
var valor = 10; — Variável valor recebe o número 10.
alert(valor); 10 Exibe o valor atual da variável: 10.
var valor = 14; — Redefine a variável valor para 14.
var msg = "Nro =" + valor; — Concatenação: msg vira "Nro =14" (string + número = string).
alert(msg); Nro =14 Exibe a mensagem "Nro =14".
var valor = "4" + 3 + 1; "431" Concatenação: começa com string "4", então tudo vira string.
var valor = 4 + 3 + "1"; "71" Primeiro soma: 4 + 3 = 7; depois concatena com string "1" → "71".
var valor = "4" + 3 - 1; 42 "4" + 3 vira "43" (string), "43" - 1 vira 42 (conversão implícita).
var valor = "30" * 2; 60 Multiplicação força conversão para número → "30" * 2 = 60.
